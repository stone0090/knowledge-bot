# Wiki 索引

> 本文件由 knowledge-bot 自动维护。每次 ingest/compile 后 `app/vault/indexer.py` 追加条目。
> 方法论见仓库 `docs/llm-wiki-method.md`，运行契约见同级 `SCHEMA.md`。

## 目录

- **Raw/**       原始来源（LLM 只读不改）
  - `articles/`     网页正文 / 公众号 / 博客
  - `notes/`        纯文本笔记 / 会议记录
  - `files/`        docx / pdf / pptx 解析文本
  - `transcripts/`  视频字幕（yt-dlp 产物）
  - `papers/`       学术论文
  - `assets/`       原始二进制附件
- **Wiki/**      LLM 编译产物（按 type 分路）
  - `entities/`     实体页（人物 / 工具 / 公司 / 项目）
  - `concepts/`     概念页（方法论 / 原则 / 术语）
  - `comparisons/`  对比页（横向对照）
  - `queries/`      `/查` 固化页
- `SCHEMA.md`    人机契约（命名 / 模板 / 标签 / 流程）
- `log.md`       最近变更时间线
